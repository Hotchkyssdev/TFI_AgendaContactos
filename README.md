# TFI_AgendaContactos
**Codigo** 
Agenda de Contactos
**Funcion**
Este programa cumple la funcion de crear una agenda de contactos, donde se le pregunta al usuario si es qué quiere añadir a una persona y de ahí le pide la informacion (Nombre, numero de telefono, email).
